# Habitat Vibecode

A Java application with a front-end.

## Structure

* `src/main/java`: Main Java source code
* `src/main/webapp`: Web application resources (HTML, CSS, JS)
* `src/test/java`: Test source code
* `pom.xml`: Maven project configuration

## Building and Running

To build the project, run:
mvn clean install

To run the application, execute the main class `com.habitat.vibecode.Main`.
